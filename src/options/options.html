+Emoji options

This will adjust the size of the emoji displayed.

Some custom fonts use characters from the Private Use Area. This options makes sure we don't break sites with such fonts. But keep in mind that some Emojis will not be shown when this is turned on (e.g. SoftBank)

If you have a native font installed on your system, +Emoji will attempt to use this instead of images ('Segoe UI Emoji' or 'Segoe UI Symbol'). Otherwise a font called 'Symbola' is used, which comes bundled with this extension.

A list of domains which should NOT display emoji. For example if you typed test.com, it would disable emojis on all subdomains including images.test.com, mail.test.com, etc. You can separate them by commas (,) and newlines (↩).
